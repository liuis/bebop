# conflux-web

This is the main package of [ConfluxWeb3.js][repo]

## Installation

### Node.js

```bash
npm install conflux-web
```

### Usage

```js 
import ConfluxWeb from 'conflux-web';

new ConfluxWeb(...);
```

## Types 

All the typescript typings are placed in the types folder.
