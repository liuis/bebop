# conflux-web-cfx

This is a sub package of [ConfluxWeb.js][repo]

This is the Cfx package to be used [ConfluxWeb.js][repo].

## Installation

```bash
npm install conflux-web-cfx
```

## Usage

```js
import {Cfx} from 'conflux-web-cfx';

const cfx = new Cfx(
    'http://127.0.0.1:4546',
    null,
    options
);
```
